A command-line tool for tracking small recurring events — coffees, workouts, moods, whatever. Each entry is a timestamp and a short tag. Data lives in a plain text file.

Built it to replace a spreadsheet habit. Used it for about four months before realizing I could do the same thing with a shell alias and a text file. Archived — but the constraint of append-only logging still informs how I think about personal data.
