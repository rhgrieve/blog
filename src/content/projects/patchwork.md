An experiment in extreme simplicity. One script, no dependencies, no config file. Point it at a folder of markdown files and it produces a folder of HTML files.

Still work in progress. The interesting constraint is what to leave out — no templating language, no plugin system, no asset pipeline. Just the transform from markdown to a readable page.
