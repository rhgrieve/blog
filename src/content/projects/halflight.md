A dark color theme designed for long sessions. The palette avoids high-contrast pairings that cause eye strain. Syntax colors are muted and meaningful — similar constructs get similar colors, so the structure of code is visible without being loud.

Available for Neovim and VS Code. Built from a small set of base hues, algorithmically adjusted for perceptual uniformity.
