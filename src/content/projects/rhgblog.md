## Overview

The rhg.sh blog is built with Astro using vanilla CSS and a hybrid static deployment. The majority of the site is statically generated, with the exception of the activity page that aggregates activity from various platforms and served from a Cloudflare Worker. 



The site you're looking at. Built with Astro, styled with plain CSS, deployed statically. No frameworks, no build-time complexity beyond what Astro provides.

## Goals

- Easy to update by dropping in markdown files
- Dark, editorial aesthetic
- Fast, minimal client-side JS
- RSS support

## Stack

- **Astro** for static generation and content collections
- **MDX** for richer content when needed
- **Plain CSS** with custom properties
- Deployed to [wherever you deploy it]
